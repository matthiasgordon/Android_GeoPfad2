Hintergrundinformation
----------------------

Die GEOpfad App wurde von den oben aufgeführten Entwicklern im Rahmen
des Moduls 'Projekte der Wirtschaftsinformatik' bei Herrn Prof. Dr. Thomas Seifert
an der Fachhochschule der Wirtschaft (FHDW) Bergisch Gladbach entwickelt.

Im Anschluss an das Modul optimierte das Entwicklerteam die Android-Applikation
für die Veröffentlichung in Kooperation mit der Stadt Bergisch Gladbach.
